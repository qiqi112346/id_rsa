视图模块
  • MapView

    二维场景视图(mapboxgl引擎)，对地图引擎进行管理，如果要对地图图层进行管理请参考Map，
    参考示例： [初始化二维场景视图]
    [ES5引入方式]：
    zondy.mapboxgl.MapView()
    [ES6引入方式]：
    import { MapView } from '@mapgis/webclient-mapboxgl-plugin'

  • Popup

    二维场景信息弹窗(mapboxgl引擎) 参考示例： [初始化二维场景视图] [ES6引入方式]：
    import { Popup } from '@mapgis/webclient-mapboxgl-plugin'
    自定义样式说明：
    zondy-popup__content 弹窗容器样式
    zondy-popup__tip 弹窗对话框箭头样式
    zondy-popup__header 弹窗头部样式
    zondy-popup__content 弹窗主体样式
    zondy-popup__footer 弹窗底部样式

  • Screenshot

    打印工具

草图编辑模块
  • SketchEditorMapboxgl

    二维Mapboxgl引擎草图编辑类

    [ES5引入方式]：
    const { SketchEditor } = zondy.mapboxgl.tool.sketch
    [ES6引入方式]：
    import { SketchEditor } from "@mapgis/webclient-leaflet-plugin"
