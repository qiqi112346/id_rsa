MapGIS WebClient for MapboxGL API

Mapbox GL主要是构建世界上最漂亮的地图，因此核心功能就是一个看字.

MapboxGL 优点

MapboxGL是最适合常规gis开发的地图，因此核心功能就是传统GIS功能.

  1. 高效矢量瓦片-真正高效实用的矢量瓦片
  2. 顶级可视化-真正顶级的可视化渲染，mapboxGL,echartGL，KeplerGl等。
  3. 高清矢量图形- 真正顶级的高清矢量图形绘制SVG，Canvas.
  4. Top级互联网技术加持 - 国内Baidu，国外Uber，Mapbox等顶级可视化巨头技术加持.
