#### 关键代码
由于echart本省是作用在百度地图上的，因此别的地图在使用的时候需要设置对应的坐标系，以leaflet举例，关键代码如下

``` javascript
var option = {
    //backgroundColor: '#000',
    title: {
        text: '10000000 GPS Points',
        left: 'center',
        textStyle: {
            color: '#fff'
        }
    },
    leaflet: {     //关键地方---1
        roam: true
    },
    series: [{
        name: '弱',
        type: 'scatterGL',
        progressive: 20000,
        coordinateSystem: 'leaflet',//关键地方---2
        symbolSize: 1,
        zoomScale: 0.002,
        blendMode: 'lighter',
        large: true,
        itemStyle: {
            color: 'rgb(20, 15, 2)'
        },
        postEffect: {
            enable: true
        },
        silent: true,
        dimensions: ['lng', 'lat'],
        data: new Float32Array()
    }]
};
```
